# Vision Voice

Vision Voice is an AI-powered prototype that helps visually impaired users by detecting objects through a camera and converting them into speech.

## Features
- Real-time object detection
- Camera access
- Voice output using text-to-speech
- Accessibility-focused design

## Technologies Used
- Python
- AI Object Detection
- Text-to-Speech

## Future Scope
- Currency detection
- Face recognition
- Navigation assistance
